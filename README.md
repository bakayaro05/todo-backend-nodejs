# Todo Backend API

A simple Todo backend built while completing  
**Backend Full Course by Smoljames**.

This project helped me understand how backend systems work and how to reason about AI-generated code.

## Tech Stack
- Node.js
- Express.js
- Prisma ORM
- SQLite
- JWT Authentication
- Docker (basics)

## Features
- User authentication using JWT
- Create, read, update, delete (CRUD) todos
- RESTful API using GET / POST routes
- Database interaction via Prisma
